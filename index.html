Part 0: Overview

For this project, I designed a website using React which allowed users to filter minecraft animals by the dimension they can be found in and whether or not they attack the player. The user can also sort the mobs by height and health.

Part 1: Implementation

To implement this functionality, I used componenets, which allowed me to pass functions and variables between javascript objects. The project has an aggregator, which counts the total height and health of the favorited mobs, as well as functionality for sorting and filtering.
